# Mobile First: Instagram

## Objetivo:
Desenvolver uma versão **mobile first** do Instagram utilizando apenas **HTML5** e **CSS3**.

## Tópicos:
Conhecimentos praticados:  
- **HTML5**;
- **CSS3**;
- **[Metodologia BEM CSS](http://getbem.com/naming/)**.
